# 세무 관련 파일 처리 가이드

## 🚀 환경 설정 (윈도우/맥 공통)

> ⚠️ **중요**: 윈도우와 맥에서 동시에 사용하려면 각 플랫폼에서 **가상환경을 별도로 설정**해야 합니다.

### 📋 필수 요구사항
- Python 3.8 이상
- pip (Python 패키지 관리자)

### 🔧 설치 방법

#### **윈도우 (Windows)**
```bash
# 1. 가상환경 생성
python -m venv venv

# 2. 가상환경 활성화
venv\Scripts\activate

# 3. 패키지 설치
pip install -r requirements.txt
```

#### **맥 (macOS)**
```bash
# 1. 가상환경 생성
python3 -m venv venv

# 2. 가상환경 활성화
source venv/bin/activate

# 3. 패키지 설치
pip install -r requirements.txt
```

### ✅ 설치 확인
```bash
python -c "import pandas, openpyxl; print('설치 완료!')"
```

> 💡 **팁**: 각 플랫폼에서 가상환경을 사용하면 서로 다른 아키텍처(x86_64 vs arm64) 문제를 피할 수 있습니다.

- 터미널에 시트 총액 확인
- `/0.967` 처리된게 I열에 입력될건데 이미 처리된거들 보고 구분 잘해서 `(전달)` 파일에 입력

<details>
<summary><strong><span style="color: gray;">원천징수 처리 케이스 (클릭하여 펼치기)</span></strong></summary>

**케이스 a) 실제로 원천징수 제하고 지급한 경우**
- 그대로 '실입금액'을 J열(지급총액)에 입력

**케이스 b) 원천징수 `/0.967` 적용**

예시: 10,000원을 지급해야 되는 경우

- **(지급예정자가 세금을 먼저 부담한 경우)**
  - 실제로 이미 3.3% 제하고 입금했다면 **9,670원**을 입력 J열(지급총액)
  
- **(고야 측에서 세금을 부담하는 경우)**
  - 실제로 3.3% 제한 금액이 10,000원이라면 지급총액은 약 **10,341원** (1만원 나누기 0.967)

</details>
